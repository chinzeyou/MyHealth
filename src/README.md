
### ⚠️ This project is built with Expo (React Native), but only the `/app` folder is included in this repository.  
It is a UI prototype and does not represent a full runnable Expo project.
If you want to try it out, you can setup your own expo, then use the repository from the '/app'.

### 🎨 Icons

This project uses the built-in **Expo vector icon library** for all UI icons.

These icons are provided through Expo and are part of the standard React Native development toolkit, ensuring consistency and performance across different devices.
